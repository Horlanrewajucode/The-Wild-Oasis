# THE-WILD-OASIS

A modern hotel management web application that enables hotel staff to efficiently manage guest check-ins and check-outs, bookings, stays, users, and pricing, all in real time.

Built with React on the frontend and Supabase as the backend (authentication, database, and real-time features).

#### Stay Management

- Manage length of stay, room assignments, and guest details

- Create and manage hotel staff accounts
